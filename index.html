01 — Vision
The Property Manager's Workshop
"Atelier" — a workshop where craft happens. Leads enter via Finn.no, are scored by AI, and serious leads receive a proactive SMS with a video tour link. An AI concierge named Oline handles the conversation.
📥

Capture

Leads from Finn.no / Google Form are ingested, normalized, and deduplicated automatically.

🧠

Score

AI scores every lead from form data. Only serious leads pass the gate and receive the first SMS.

💬

Engage

AI concierge handles Q&A via SMS — answering questions, sharing video tours, and pushing Calendly.

🤝

Convert

Soft commitments collected, units held first-confirmed-wins, then handoff to credit check and contract.

V1 → V2: V1 was a web chat sandbox for validating AI prompts. V2 is a production system where AI initiates contact via SMS and the PM observes/intervenes through a dashboard.

04 — SMS Experience
Natural Conversation via SMS

Debounce-Based Replies

Unlike rigid chatbots, Atelier mimics real human texting. When a tenant sends a message, a 30-second timer starts. More messages reset it. When it fires, the AI responds to everything at once.

Hard cap: if the first unresponded message is older than 2 minutes, respond immediately.

10 — Delivery Roadmap
Phased Rollout
Phase 0

Auth + Security Foundation

JWT auth, webhook verification, rate limiting, audit logging, CORS lockdown.

Phase 1

Backend Infrastructure

Haiku/Sonnet split, channel messaging, Google Sheets ingestion, scoring gate, templated first message, listing snapshots.

Phase 2

PM Dashboard (Strangler Pattern)

LeadList + attention queue, ConversationThread, PMComposer + takeover controls, retire old chat UI.

Phase 3

Scoring Refinement

Unified LeadScoring.md with preliminary + conversation modes. Four weighted sub-scores. Behavioral signals injection.

Phase 5

Contract Decision Guide

25-topic SMS-ready contract quick reference (ContractGuide.md) injected into Oline's prompt. Pre-computed answers with per-topic anti-patterns.

Phase 6

SMS Simulator via Rova

Rova backend + UI, simulator adapter, character enforcement, debounce auto-reply, E2E test script.

Phase 4

MCP Prompt Streamlining

Prompt caching for static sections. Eliminate static listing file dependencies.

11 — Compliance & Risk
Guardrails

Compliance

SMS Consent

Required checkbox on Google Form. No SMS without consent_sms = true.

STOP Handling

Keyword detection, immediate suppression, logged. Checked before every send.

GDPR

E.164 normalization. Right to deletion for Norwegian tenants.

Audit Trail

All PM actions, AI decisions, state transitions logged.

Risk Mitigation

AI Quality

Test Haiku with 5 conversations before go-live. Sonnet fallback via env var.

Cost Control

Pling at 0.39–0.55 NOK/msg. Budget 20 msgs/lead = NOK 8–11/lead.

Scoring False Negatives

Filtered leads visible in dashboard. Manual override with audit logging.

SMS Testing

Simulator routes to Rova virtual phone. Toggle SMS_CHANNEL_MODE for zero-code switching.
